    Major Fixes (1 added on Monday, April 5th)
  1. Open lines now showing while in a specific area.
  2. Using tab movement now shows the correct display.
  3. Expanded events on the large calendar view now shrink when zooming out or moving to Home.

  6. URL navigation is back to working. The following words can follow "http://dev.lapcat.org/": catalog, give, account, (any area, i.e., home) or (any tag name, i.e., golf). Navigation via the URL bar is not necessary to navigate within LAPCAT.
  7. Browse searching (clicking on an author's name, library location, date, etc.) back to working. This should now be working for both adding to and removing from a search.
  8. Start-Up/Help message box is now anchored to the bottom of the page (like Facebook) and is more robust and intuitive than before. The status log message box (right-side) is for development purposes only.

  10. The IE bug that prevented material cover images from showing has inadvertently been fixed.
  11. Tweaked the Start box (formerly Start-Up/Help) to include a LAPCAT suggestion as to what you should do. By default, LAPCAT will suggest you log in or create an account. Also included in the Start box are several common catalog-related links as well as a simple search for the catalog. Once logged-in, the suggestions and links will change.
  12. Made a copy of the old HTML layout and rearranged the HTML elements in it for a cleaner, more simplistic look and feel.
  13. The hiring area should now be working.
  14. The search menu (formerly left-hand side) is now on top bar (middle to right-hand side.)
  15. Added digital audio books (tag 159 - digital audiobook) to both client and server pages. Also, added one digital audio book to the development server for testing.
  16. Added "all materials" (no tag) to both client and server pages. The server will continue to suggest materials by material type at random - "all materials" is not a random choice for it. The user must choose to search "all materials". "All materials" is needed in order to perform a similar search.
  17. Promotions are back to working. They should automatically refresh every 30 seconds.
  18. Searches that return no results now offer suggestions to alter the search.
  19. Several searches have been fixed. Adding and removing tags should now affect all constructs (individual displays).
  20. Similar search should now be working. Similar searches automatically change the material type to "all materials". (Note: Currently, there is no way to remove the similar search once activated.)
  21. Materials' similar search (name/title search) should now be working.
  22. Added a way to remove a similar search.
  23. Pagination is back to working - at least with Materials.
  24. Fixed a bug that was causing event data to not be removed from the calendar when applying new event data.
  25. Set the pagination buttons container to a fixed width. This solves the problem of the horizontal scrollbar appearing while not at Home.
  26. Specific news and event IDs can now be used in URL navigation. For example, "http://dev.lapcat.org/events/specific/1465" will take the user to event ID 1465. This is useful for third-party site content sharing, like with Facebook.
  27. All lists and open line displays should now look and feel the same.
  28. Modified the calendar to work with light/dark themes.

  30. Pagination should be working for all areas.
